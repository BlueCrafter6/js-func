Using js-functions

js-functions was designed to be simple and easy to use

Conversion Chart

This should help you convert your long element selector code to some short and clean element selector code (using js-func)

Original js-functions
document.getElementById() getID()
document.getElementsByName() getName()
document.getElementsByTagName() getTagName()
document.getElementsByClassName() getClass()

* More will be added soon

Function Parameters

getID(id: string)

Grabs the element with the given id

Returns: HTMLElement

getName(name: string)

Grabs the element(s) with the given name

Returns: Array[ ]

getTagName(tag: string)

Grabs the element(s) with the given tag name

Returns: Array[ ]

getClass(_class: string)

Grabs the element(s) with the given class name

Returns: Array[ ]
